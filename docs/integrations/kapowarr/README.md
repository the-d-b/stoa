**Category:** Comics & Manga | **Status:** 🔶 Need Testing | **Polling:** 30 min

## Integration

**Secret format:** Plain API key

> Kapowarr → Settings → API Key

**URL required:** Required

**Example URL:** `http://kapowarr:5656`

### Setup

1. Kapowarr → Settings → copy API Key
2. Admin → Secrets → New: paste the key
3. Admin → Integrations → New: type Kapowarr, URL `http://kapowarr:5656`, secret
4. Admin → Panels → New: type Kapowarr

---

## Panel

Western comics/volume manager showing collection stats, download progress, volume library, active download queue, and a cover strip.

### Height behavior

| Height | What you see |
|---|---|
| 1x | Volume count · issue download % · monitored count · queue depth |
| 2-3x | Stats + scrollable volume list |
| 4x+ | Cover strip + stats + volume list + active download queue |
