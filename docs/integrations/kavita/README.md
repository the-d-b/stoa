## Integration

**Secret format:** Plain API key

> Kavita -> your username (top-right) -> User Settings -> API Key

**URL required:** Required

**Example URL:** `http://192.168.1.10:5000`

### Setup

1. Kavita -> your username (top-right) -> User Settings -> copy API Key
2. Admin -> Secrets -> New: paste the key
3. Admin -> Integrations -> New: type Kavita, URL, secret
4. Admin -> Panels -> New: type Kavita

---

## Panel

Series count, total files, library list, and a recently-added series strip with cover thumbnails.

### Height behavior

| Height | What you see |
|---|---|
| 1x | Series/file counts |
| 2-3x | Library list + recent series strip |
| 4x+ | Full stats + cover grid |
