# Bookmarks

**Category:** Productivity | **Status:** Tested | **Requires integration:** No - data stored locally in Stoa

---

## Panel

Visual bookmark tree displayed as a panel. Bookmarks are organized into folders and sub-folders, each optionally with a custom icon. Clicking opens the URL.

### Height behavior

| Height | What you see |
|---|---|
| 1x | Folder list + quick-access links |
| 2-3x | Expanded folder tree |
| 4x+ | Full bookmark tree with icons |
