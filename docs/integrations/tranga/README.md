# Tranga

**Category:** Comics & Manga | **Status:** 🔶 Need Testing | **Polling:** 30 min

---

## Integration

**Secret format:** Blank (no auth required by default)

> Tranga runs unauthenticated by default. Leave the API key field blank.

**URL required:** Required

**Example URL:** `http://tranga:9898`

### Setup

1. Admin → Integrations → New: type Tranga, URL `http://tranga:9898`, API key blank
2. Admin → Panels → New: type Tranga

> If you have configured an API key in Tranga (optional), paste it in the API key field.

---

## Panel

Manga library manager showing your collection, publication status (Ongoing / Completed / Hiatus), active chapter downloads, and a cover strip.

### Height behavior

| Height | What you see |
|---|---|
| 1x | Manga count · active download count |
| 2-3x | Stats + active downloads + manga library list |
| 4x+ | Cover strip + stats + manga list + active downloads column |
