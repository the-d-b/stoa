# Mylar3

**Category:** Comics & Manga | **Status:** 🔶 Need Testing | **Polling:** 30 min

---

## Integration

**Secret format:** Plain API key

> Mylar3 → Settings → Web Interface → Enable API → copy API Key

**URL required:** Required

**Example URL:** `http://mylar3:8090`

### Setup

1. Mylar3 → Settings → Web Interface → enable API, copy the API Key
2. Admin → Secrets → New: paste the key
3. Admin → Integrations → New: type Mylar3, URL `http://mylar3:8090`, secret
4. Admin → Panels → New: type Mylar3

---

## Panel

Comic/manga series library with wanted issues (missing), upcoming release calendar, and a cover strip of your series.

### Height behavior

| Height | What you see |
|---|---|
| 1x | Series count · wanted count · upcoming count |
| 2-3x | Stats + wanted issues list (or upcoming if nothing wanted) |
| 4x+ | Cover strip + stats + wanted issues column + upcoming releases column |
